# Blog App

A full stack blog web app built with Python/Flask and Bootstrap, covering topics on web development and cybersecurity.

## Tech Stack
- Python/Flask
- HTML, CSS, Bootstrap
- Jinja2 Templates
- npoint (JSON API)

## Features
- Dynamic blog posts served from a JSON API
- Covers web development and cybersecurity topics
- Responsive design with Bootstrap

## How to Run Locally
1. Clone the repo
   git clone https://github.com/YOUR_USERNAME/blog-app.git
2. Install dependencies
   pip install -r requirements.txt
3. Run the app
   python main.py
